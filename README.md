# Policy AI Assistant (Local)

## Features
- Natural language chatbot
- MySQL policy query
- Local LLM via Ollama
- Streamlit dashboard
- Finance-safe local deployment

## 1. Install Ollama
https://ollama.com

ollama pull deepseek-coder

## 2. Install Requirements
pip install -r requirements.txt

## 3. Configure DB
Edit config.py

## 4. Run App
streamlit run app.py

## 5. Open Browser
http://localhost:8501
